# KlipperScreen Theme Creator

A GTK 3 based visual theme creator for KlipperScreen themes.

Current prototype features:

- WSL2/WSLg development environment
- GTK 3 application window
- KlipperScreen-style preview area
- Separate editor CSS and preview CSS
- Clickable preview elements
- Inspector panel showing selected UI role
- Live editing for background color, opacity, text color, border color, border width, and border radius

Dynamic background compatibility is planned for generated themes that target the modified KlipperScreen build with BackgroundManager support.
